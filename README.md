# Zambia Languages
## Major Languages
Zambia consists of 7 languages spoken by the majority of the people. These are:
1. Bemba or Icibemba
2. Nyanja or Chinyanja
3. Kaonde or Kikaonde
4. Silozi, Lozi or Rozi
5. Lunda or Chilunda
6. Luvale
7. Tonga
